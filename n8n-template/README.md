# Template do fluxo Primo (n8n)

Este arquivo contém uma versão pública e sanitizada do fluxo do agente Primo.

O JSON representa a estrutura do agente em produção, com:
- credenciais removidas
- endpoints internos anonimizados
- dados reais excluídos

Use este template apenas como referência de arquitetura e organização de fluxo.
