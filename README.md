# Primo – Agente de IA Operacional do Obra Prima

Primo é um agente de IA desenvolvido para executar operações reais dentro do sistema Obra Prima - ERP de Gestão de Obras.

Ele recebe solicitações em linguagem natural, entende a intenção do usuário e executa ações diretamente no sistema, automatizando rotinas operacionais da obra.

Este repositório apresenta uma versão pública e sanitizada do projeto.

## O que o Primo faz

O Primo executa, de ponta a ponta:

- Pré-lançamentos financeiros no sistema Obra Prima
- Solicitações de material e compras
- Rotinas do dia a dia da obra
- Validação de dados e confirmações com o usuário
- Integração direta com APIs e banco de dados do sistema

Não é um chatbot demonstrativo.
É um agente operacional em ambiente real.

## Exemplos de uso

- “Lançar um pré-lançamento de R$ 1.200 para material elétrico”
- “Criar uma solicitação de material para a obra X”
- “Consultar ou atualizar informações operacionais da obra”

## Tecnologias utilizadas

- n8n
- Modelos de linguagem (LLMs)
- APIs REST
- PostgreSQL / SQL Server
- Redis
